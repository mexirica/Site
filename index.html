Minhas Habilidades

Python

Python é uma linguagem de programação extremamente simples e versátil, pois, como sua sintaxe é moderna e objetiva, ela foi elaborada para que as pessoas desenvolvedoras escrevam instruções com menos linhas de código.

HTML5

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. Ela traz novas funções como as tags video, audio e elemento canvas e será o novo padrão para HTML, XHTML e HTML DOM.

CSS

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.
